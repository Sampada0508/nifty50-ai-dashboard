## 🚀 Features

- 📊 Real-time NIFTY 50 data analysis  
- 🔮 7-day predictions (High, Low, Expected)  
- 📉 Risk & confidence analysis  
- 📊 Interactive charts using Plotly  
- 🤖 Multiple ML model comparison  

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Pandas  
- NumPy  
- Scikit-learn  
- Plotly



## 📸 Project Preview

- 📊 Interactive dashboard for NIFTY 50 predictions  
- 📈 Real-time charts with technical indicators  
- 🤖 AI-based prediction results with confidence levels  

## ▶️ How to Run

- Clone the repository  
  git clone https://github.com/Sampada0508/nifty50-ai-dashboard.git  

- Navigate to project folder  
  cd nifty50-ai-dashboard  

- Install dependencies  
  pip install -r requirements.txt  

- Run the application  
  streamlit run app.py
